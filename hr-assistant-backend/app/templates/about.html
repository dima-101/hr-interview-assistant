О проекте

HR Interview Assistant — корпоративный веб-сервис для проведения первичных HR-интервью, управления вакансиями и подготовки структурированных отчётов по итогам собеседований.

Решение предназначено для стандартизации первичного отбора кандидатов, снижения нагрузки на HR-специалистов и повышения качества предварительной оценки соискателей по заданным профилям вакансий.

Сервис объединяет карточки вакансий, сценарий интервью, сбор ответов кандидата и подготовку итогового отчёта, который может использоваться специалистом по подбору персонала для дальнейшего принятия решения.

Архитектура проекта предусматривает расширение набора вакансий, подключение внешней LLM-модели, развитие логики оценки кандидатов и интеграцию с внутренними HR-процессами компании.
