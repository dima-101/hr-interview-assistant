# Candidate Evaluation Template

## О документе

Этот шаблон используется в проекте **HR Interview Assistant** для оформления итогового отчёта по кандидату после проведения интервью.

Шаблон универсальный: его можно применять к любой вакансии, если ассистент подставляет конкретные данные из соответствующей карточки профиля.

## Инструкция по использованию шаблона

1. Ассистент должен подставлять в этот шаблон данные только на основе:
   - ответов кандидата;
   - карточки выбранной вакансии;
   - общей базы знаний компании.

2. Нельзя придумывать опыт, навыки или достижения кандидата, если они не подтверждены в интервью.

3. Если данных недостаточно, это должно быть прямо указано в соответствующем разделе.

4. Баллы и критерии должны соответствовать именно той рубрике, которая указана в карточке выбранной вакансии.

5. Финальный вывод должен быть нейтральным, деловым и основанным на фактах.
